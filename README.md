# NVIDIA 动量交易策略 - 2025版

## 📊 项目简介

这是一个基于2025年最新动量交易理念开发的量化策略，专门针对NVIDIA (NVDA) 股票的波动特性设计。策略集成了多个经过市场验证的技术指标，包括移动平均线交叉、RSI、MACD和成交量确认。

## 🎯 策略核心

### 技术指标组合
1. **双移动平均线系统** (50日/200日)
   - 金叉/死叉作为主要趋势判断
   - 基于2025年市场研究的最优参数

2. **RSI动量指标** (14周期)
   - 超买区域: RSI > 70
   - 超卖区域: RSI < 30
   - 用于捕捉反转机会

3. **MACD趋势确认**
   - 快线: 12日EMA
   - 慢线: 26日EMA
   - 信号线: 9日EMA
   - 金叉/死叉作为辅助确认

4. **成交量突破验证**
   - 20日成交量移动平均
   - 确保价格突破有足够的市场参与度

### 交易信号逻辑

**买入条件** (满足任一即可):
- 50日均线上穿200日均线 (金叉)
- RSI从超卖区反弹 + 上升趋势确认
- MACD金叉 + RSI未超买 + 成交量放大

**卖出条件** (满足任一即触发):
- 50日均线下穿200日均线 (死叉)
- RSI进入超买区域 (>70)
- MACD死叉
- 动量转负 + 成交量萎缩

## 📊 性能指标说明

### 夏普比率 (Sharpe Ratio)
- 衡量风险调整后收益
- > 1.0: 良好
- > 2.0: 优秀
- > 3.0: 卓越

### 最大回撤 (Max Drawdown)
- 衡量最大损失幅度
- < 10%: 低风险
- 10-20%: 中等风险
- > 20%: 高风险

## ⚠️ 风险提示

1. **历史表现不代表未来**: 回测结果仅供参考
2. **市场波动**: NVIDIA波动率高达40%+，需要良好的风险控制
3. **滑点成本**: 实盘交易会有交易成本和滑点
4. **黑天鹅事件**: 策略无法预测突发重大事件

## 🔧 参数调优建议

可以在代码中调整以下参数:
- `period`: 历史数据周期 (默认1年)
- `initial_capital`: 初始资金 (默认$10,000)
- SMA周期: 50日和200日 (可改为其他组合)
- RSI超买/超卖阈值: 70/30 (可调整)
- MACD参数: 12/26/9 (快/慢/信号线)
